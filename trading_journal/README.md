## 📋 使用說明

### 每日流程：
1. 複製 `TEMPLATE.md` 為 `YYYY-MM-DD.md`
2. 運行預市場掃描
3. 記錄學習內容
4. 記錄交易決策
5. 更新本索引

### 命令：
```bash
# 複製模板
cp trading_journal/TEMPLATE.md trading_journal/$(date +%Y-%m-%d).md

# 查看今日日記
cat trading_journal/$(date +%Y-%m-%d).md
```
