# 香港AI生活助手 - 規格文件

## 1. Concept & Vision

**「香港日常」** — 一個真係為香港人而設嘅AI生活助手。

唔係另一個旅行App，係你每日都會用嘅搭公交、天氣、搵地方嘅智能伙伴。用廣東話思考，用廣東話回答，就係香港人嘅助手。

## 2. Design Language

### 美學方向
香港霓虹夜景 + 現代簡潔介面嘅混合風格。參考：深夜茶記🏮 + MTR指示牌🟡

### 色彩系統
```
Primary:     #FF6B35 (香港橙)
Secondary:   #1A1A2E (深藍黑)
Accent:      #FFD93D (MTR黃)
Background:  #0F0F1A (深夜黑)
Surface:     #1E1E2E (卡片背景)
Text:        #FFFFFF (白)
Text-muted:  #A0A0B0 (灰字)
Success:     #4ADE80 (綠)
Warning:     #FBBF24 (黃)
Error:       #EF4444 (紅)
```

### 字體
- **標題:** Noto Sans HK Bold
- **內文:** Noto Sans HK Regular  
- **英文:** Inter

### 動效哲學
- 滑動流暢如搭MTR扶手梯
- 卡片展開如茶餐廳出餐
- 重要通知用霓虹閃爍效果

## 4. Features & Interactions

### 4.1 天氣模組
- **核心數據:** 溫度、濕度、紫外線、空氣質素
- **香港特色:**
  - 「今日著咩？」- 根據溫度+濕度建議
  - 「適合晾衫？」- 回南天警告
  - 颱風路徑圖（連結天文台）
- **卡片互動:** 向上滑睇逐小時預報

### 4.2 交通模組
- **核心功能:**
  - MTR即時到站（東涌線/荃灣線/觀塘線等）
  - 小巴到站時間估算
  - 巴士到站（九巴/城巴）
- **輸入方式:** 語音「東涌站幾耐到？」/ 文字/最近站點
- **輸出格式:** 
  ```
  🟡 荃灣線 → 中環
  下一班: 2分鐘
  兩班後: 7分鐘
  ```

### 4.3 探索模組
- **「街坊眼」** - 香港人先知嘅地方
- **分類:**
  - 🍜 隱世食肆（唔係遊客嗰啲）
  - 🏛 社區故事（街市傳說/老店歷史）
  - 🌳 秘密花園（行山徑/天台農場）
- **呈現方式:** 卡片+圖片+一句點評

### 4.4 AI對話
- **啟動方式:** 底部輸入框 / 語音按鈕
- **語言:** 廣東話為主，可混合英文
- **能力範圍:**
  - 天氣查詢
  - 交通路線
  - 地方推薦
  - 香港資訊问答

## 5. Component Inventory

### 5.1 天氣卡片
- **預設:** 顯示溫度+天氣圖示+一句描述
- **展開:** 顯示逐小時/逐日預報
- **加載:** 骨架屏 + 閃爍效果

### 5.2 交通到站卡
- **預設:** 路線名+下一班時間
- **狀態:** 
  - 🟢 正常 (3分鐘內)
  - 🟡 稍等 (3-10分鐘)  
  - 🔴 久等 (>10分鐘)
- **按鈕:** 收藏路線 / 分享

### 5.3 探索地方卡
- **呈現:** 圖+標題+分類標籤+街坊評分
- **點讚:** 👍 vs 幫趁 💰
- **點擊:** 展開詳細介紹

### 5.4 AI對話泡
- **用家:** 右側，橙底白字
- **AI:** 左側，深灰底白字，帶🐉標誌
- **加載中:** 三點跳動...

### 5.5 導航底部
- 5個圖標：首頁/交通/探索/對話/設定
- 當前頁面高亮（MTR黃色）
- 快速滑動切換

## 6. Technical Approach

### 前端
- **框架:** React Native (Expo)
- **語言:** TypeScript
- **狀態管理:** Zustand
- **導航:** React Navigation

### 後端整合
- **天氣API:** Open-Meteo (免費) + 香港天文台開放數據
- **交通API:** MTR Open API / LRT API
- **AI引擎:** GPT-4o-mini (廣東話優化)

### 數據存儲
- **用戶偏好:** AsyncStorage
- **收藏路線:** 本地 + iCloud備份
- **對話歷史:** 本地SQLite

### 語音
- **輸入:** iOS Speech Recognition
- **輸出:** 本地TTS / ElevenLabs廣東話
